# Instalation
- first run ``` composer update ``` using terminal
- configure database in .env.development
``` 
DB_CONNECTION=mysql
DB_HOST=localhost
DB_DATABASE=tasik_fun
DB_USERNAME=root
DB_PASSWORD=asobiasobase
```
- database folder in  [database/tasik_fun.sql](database/tasik_fun.sql)
